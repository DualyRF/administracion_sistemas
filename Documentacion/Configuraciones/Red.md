Configuracion de la red.
Las maquinas se configuraron con *2 adaptadores* para poder acceder a la red y para poder hacer pings entre ellas.

Nodo 1: Servidor de Linux:
Se le asigno la IP estatica de *192.168.50.100*
Nodo 2: Servidor de Windows:
Se le asigno la IP estatica de *192.168.50.20*
Nodo 3: Cliente:
Se le asigno la IP estatica de *192.168.50.30*

El nombre utilizado para la red interna fue *red_sistemas*
